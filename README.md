# RUSH.PYTHON

Repositório criado para registrar minha evolução em Python ao longo de 1 mês de prática.

Aqui estão exercícios, testes e códigos que desenvolvi durante esse período, com foco em lógica de programação e fundamentos da linguagem.

## Objetivo
Mostrar TUDO (ou quase) do que venho produzindo, acompanhando minha evolução na prática.


Minha meta é evoluir para projetos mais completos, como:

- jogos com IA
- dashboards com Streamlit ou Dash
- gráficos e visualização de dados
- sistemas com filtros e dados em tempo real
- softwares baseados em ideias próprias

Tecnologias e Skills
- Python
- Django
- TDD (Testes com unittest)
- Type Hints (Tipagem de dados)
- Structural Pattern Matching
- Design Patterns (POO / GOF)
- Regex (Expressões regulares)
